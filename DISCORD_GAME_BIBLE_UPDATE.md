# 📖 SHADOW SUPPLY DEVELOPMENT UPDATE

## Official Game Bible Established

Shadow Supply now has a canonical Game Bible defining the release vision for the entire project.

The Bible locks in the game's core direction:

- A low-level counterfeiter starting premise
- Relationship-driven access to suppliers, buyers, employees, and specialists
- Rapport, trust, respect, loyalty, district reputation, industry reputation, and faction standing
- Manual interactive production for every recipe
- Physical power, equipment, logistics, storage, and deliveries
- Branching factions and consequences
- A persistent criminal-career sandbox designed for 300+ hour saves
- A handcrafted island metropolis with dynamic district and faction state
- New worlds that can produce materially different criminal careers
- A story that feeds the sandbox instead of ending the game

The Bible also establishes a formal change process. Future changes to the core game or map must update the Bible and its changelog so the release vision is never silently lost during development.
