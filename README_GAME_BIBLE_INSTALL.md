# Shadow Supply Game Bible v1.0.0

This package establishes the canonical release vision for Shadow Supply.

## Files

- `Documentation/SHADOW_SUPPLY_GAME_BIBLE.md`
  - Main design authority
- `Documentation/GAME_BIBLE_CHANGELOG.md`
  - Approved amendment history
- `Documentation/GAME_BIBLE_CHANGE_REQUEST_TEMPLATE.md`
  - Template for future core changes
- `Documentation/Images/production_progression_concept.png`
  - Initial production-branch concept supplied during planning
- `CHATGPT_START_HERE.md`
  - Updated required reading order
- `PROJECT_CONTEXT.md`
  - Updated project-level context
- `CURRENT_STATUS.md`
  - Updated to the latest confirmed design and implementation state

## Install

Extract into the repository root and replace the three root Markdown files.

No Unity project setup action is required.

## Future rule

Any approved change to the core game, map, progression philosophy, relationship structure, production rule, or replayability target must update both:

- `Documentation/SHADOW_SUPPLY_GAME_BIBLE.md`
- `Documentation/GAME_BIBLE_CHANGELOG.md`
