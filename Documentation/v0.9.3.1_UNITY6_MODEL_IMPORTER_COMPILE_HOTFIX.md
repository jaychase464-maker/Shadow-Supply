# v0.9.3.1 — Unity 6 Model Importer Compile Hotfix

## Fixed

Removed the obsolete and read-only Unity API assignment:

`ModelImporter.importMaterials = false`

The visual integration setup already uses the supported Unity 6 API:

`ModelImporter.materialImportMode = ModelImporterMaterialImportMode.None`

## Installation

1. Exit Play Mode.
2. Close Unity.
3. Extract this ZIP into the Unity project root.
4. Replace the prompted script.
5. Reopen Unity and wait for compilation.
6. Run:
   `Shadow Supply → Setup → Apply Milestone 9A Visual Model Integration`
7. Then run:
   `Shadow Supply → Validation → Validate Milestone 9A Visual Model Integration`

No original Milestone 9A setup rerun is required.
