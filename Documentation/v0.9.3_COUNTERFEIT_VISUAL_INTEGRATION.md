# v0.9.3 — Counterfeit Production Visual Integration

## Purpose

This update replaces the Milestone 9A prototype blocks with the complete set of
user-supplied Meshy models while preserving the confirmed interaction logic,
drop-zone hotfix, labels, mistake handling, relationships, and save schema 10.

## Integrated models

1. Imprint Press
2. Blank Note Stock
3. Pigment Capsule
4. Security Film
5. Printed Replica Sheet
6. Cutting Mat
7. Calibration Toolkit
8. Cut Note Stack
9. Packaging Set
10. Replica Note Bundle

## Integration behavior

- Imported FBX files use Unity model import optimization and medium mesh
  compression.
- Large source textures were reduced before packaging.
- Base-color and normal textures are connected to URP Lit materials.
- Security Film uses a semi-transparent blue-green material.
- Every imported object is automatically oriented, uniformly scaled, centered,
  and placed on the tabletop.
- The prototype press body and cutting-mat blocks are hidden.
- Existing work zones remain as translucent interaction guides.
- Printed Replica Sheet and Cut Note Stack now appear during their correct
  production steps.
- Inventory, held-item, dropped-item, production, and completed-output visuals
  use the imported models.
- The finished output sitting on the workbench now uses the Replica Note Bundle.

## Installation

1. Exit Play Mode.
2. Close Unity.
3. Extract the ZIP into the Shadow Supply Unity project root.
4. Replace every prompted file.
5. Reopen Unity.
6. Allow Unity to import and compress the ten FBX models. This can take several
   minutes on the first launch.
7. Open:

   `Assets/_Project/Scenes/Development/Dev_Playground.unity`

8. Run:

   `Shadow Supply → Setup → Apply Milestone 9A Visual Model Integration`

9. Run:

   `Shadow Supply → Validation → Validate Milestone 9A Visual Model Integration`

Expected result:

`COUNTERFEIT VISUAL INTEGRATION READY`

Do not rerun the original Milestone 9A setup menu.

## Test flow

1. Enter Play Mode.
2. Start a fresh counterfeit production run.
3. Confirm Blank Note Stock is a paper stack rather than a yellow block.
4. Confirm the Pigment Capsule and Security Film use the imported models.
5. Confirm the large central block is replaced by the Imprint Press.
6. Complete PRINT and confirm the Printed Replica Sheet appears.
7. Move it onto the imported Cutting Mat.
8. Click the imported Calibration Toolkit.
9. Confirm the imported Cut Note Stack appears.
10. Add the imported Packaging Set.
11. Seal and collect the imported Replica Note Bundle.
12. Select, drop, and hold the finished bundle to verify inventory visuals.
13. Deliver it to Mara and verify the buyer flow remains functional.
14. Save during production and verify the correct models return after loading.

## Notes

The importer attempts to correct Meshy orientation automatically by testing
several 90-degree rotations and selecting the one that best matches the expected
shape. Minor per-model rotation or scale calibration may still be needed after
seeing the assets inside the actual Unity scene.
