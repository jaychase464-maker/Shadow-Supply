# Milestone 5 Imported Model Sources

The following user-supplied archives were incorporated into the Milestone 5 package:

- `workbench-low-poly.zip`
- `metal_pallet_racking_wbslaikga_mid.zip`
- `med_office_desk_fbx.zip`
- `cc0-camera.zip`
- `cc0-keypad-door-lock.zip`

Selected FBX geometry and required texture maps are stored under:

`Assets/_Project/Art/Imported/UserFurniture/`

The utility cabinet remains generated placeholder geometry because the available wardrobe archive is substantially heavier and should receive a dedicated optimization and material pass before inclusion.

Before commercial release, verify every source archive's license, attribution requirements, redistribution rights, and allowed modifications.
